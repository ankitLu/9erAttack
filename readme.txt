Game: 9erAttack

Members:
	Myung Park
	Ankit Luthra
	Cameron Fray
	Javis Belton

How to play: 	Maneuver using Directional Keys. Aim with Mouse, Mouse1 Click to shoot, hold down for repeated fire.
		Achieve required score per level to advance.
		Defeat final boss on last stage to win the game.

Cheat Codes:	Q to add 99 lives.
		E to add 99 score.
